# Python数据类型

### 1.简单数据类型

- **int（整数）**

  - 表示整数，如 `0`、`100`、`-5`、`0b101`（二进制）、`0o17`（八进制）、`0xFF`（十六进制）

  - ```
    a = 10
    b = 3
    print(type(a))  # <class 'int'>
    ```

- **float（浮点数）**

  - 表示小数，如 `3.14`、`2.0`、`-0.5`，也支持科学计数法 `1e3`（即 1000）

  - ```
    pi = 3.14159
    print(type(pi))  # <class 'float'>
    ```

- **bool （布尔型）**

  - 只有两个取值：`True`（真）和 `False`（假），常用于条件判断。

  - ```
    is_ok = True
    print(type(is_ok))  # <class 'bool'>
    ```

- **complex（复数）**

  - 表示数学中的复数，形式为 `a + bj`，其中 `a` 是实部，`b` 是虚部，`j` 是虚数单位。

  - ```
    c = 3 + 4j
    print(type(c))  # <class 'complex'>
    print(c.real)  # 3.0(实部)
    print(c,imag)  # 4.0(虚部)
    ```

### 2.复合数据类型

- **str（字符串）**

  - 由字符组成的有序序列，用单引号 `' '`、双引号 `" "` 或三引号 `''' '''`/`""" """` 包裹

  - 不可变（修改会生成新字符串），支持索引、切片

  - ```
    s = "Hello,World!"
    print(s[0])  # 'H' (索引)
    print(s[1:5])  # 'ello' (切片)
    ```

- **list(列表)**

  - **有序、可变**的序列，用方括号 `[]` 包裹，元素用逗号分隔，可存储不同类型数据

  - 支持增删改查操作

  - ```
    lst = [1, "apple", 'a', "day", True, 3.14, ['a', 'b']]
    lst.append(5)  # 添加元素
    lst[1] = "banana"  # 修改元素
    print(lst)  # [1, "apple", 'a', "day", True, 3.14, ['a', 'b']]
    ```

- **tuple(元组)**

  - **有序、不可变**的序列，用圆括号 `()` 包裹，元素用逗号分隔

  - 一旦创建就不能修改，适合存储固定数据

  - ```
    t = (1, 2, 3)
    print(t[0])  # 1
    # t[0] = 10  # 报错：元组不可变（区别于列表）
    ```

- **dict(字典)**

  - **无序**的键值对集合，用花括号 `{}` 包裹，格式为 `key: value`

  - 键必须唯一且不可变，值可以是任意类型

  - ```
    d = {"name": "Alice", "age": 20, "is_student": True}
    print(d["name"])  # 'Alice'
    d["age"] = 21  # 修改值
    ```

- **set(集合)**

  - **无序、不重复**的元素集合，用花括号 `{}` 包裹（空集合用 `set()`）

  - 常用于去重、交集 / 并集 / 差集等数学运算

  - ```
    s1 = {1, 2, 2, 4}
    s2 = {2, 4, 5}
    print(s1)  # {1, 2, 4} (自动去重复)
    print(s1 & s2)  # {2, 4} (交集)
    print(s1 | s2)  # {1，2，4，5} （并集）
    ```



### 3.核心区别总结

| 类型                   | 可变性 | 有序性       | 存储特定     | 经典用途             |
| ---------------------- | ------ | ------------ | ------------ | -------------------- |
| int/float/Bool/complex | 不可变 | -            | 单一值       | 数值计算，逻辑判断   |
| str                    | 不可变 | 有序         | 字符序列     | 文本处理             |
| list                   | 可变   | 有序         | 任意类型元素 | 频繁修改的有序数据   |
| tuple                  | 不可变 | 有序         | 任意类型元素 | 固定数据，函数返回值 |
| dict                   | 不可变 | 有序（3.7+） | 键值对       | 映射关系，配置关系   |
| set                    | 可变   | 无序         | 唯一元素     | 去重，集合运算       |

**可变&不可变**

- 可变：数据**创建后就无法直接修改原内容**，如果尝试修改，会生成一个**全新的对象**，原对象保持不变
- 不可变：数据**可以直接在原对象上修改内容**，修改后对象的内存地址不变，不会生成新对象

**有序&无序**

- 有序：数据的元素**有固定的排列顺序**，每个元素都对应唯一的**索引下标**（从 0 开始），可以通过索引精准获取、切片操作，新增元素的位置固定，不会随机变化

- 无序：元素**没有固定的排列顺序**，**不支持索引 / 切片操作**，遍历或查看时元素顺序可能和定义时不一致，仅关注元素本身，不关注位置

  > 重要补充：Python 3.7 及以上版本，**字典（dict）变为有序**（保留元素插入顺序），这是官方标准特性，日常开发按有序处理即可。
